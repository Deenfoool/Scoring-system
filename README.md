**Discord Activity Manager** — это специализированный инструмент для автоматизации подготовки отчетов на повышение во фракциях Discord. Программа берет на себя рутину: от хранения ссылок до умного распределения баллов с учетом строгих лимитов.

## ✨ Ключевые фишки

* **🌐 Современный интерфейс**: Построен на базе `PyWebView`, что дает плавный и красивый UI в стиле темной темы Discord.
* **🧠 Умное "Правило 50%"**:
  * Если новость дает баллов меньше, чем 50% от цели — она зачисляется полностью.
  * Если новость "жирная" и превышает 50% от цели — алгоритм автоматически урезает её до максимально допустимого лимита.
* **🛠 Полуавтоматический режим**: Программа предлагает список доступных новостей, а вы решаете, какую выгодно потратить сейчас, а какую приберечь для следующего ранга.
* **🔗 Умный парсинг**: Просто копируйте ссылки из Discord пачкой — менеджер сам определит тип активности по ID канала.
* **↩️ Система отката**: Одной кнопкой можно вернуть все ссылки из последнего отчета обратно в базу, если вы передумали его подавать.

## 🚀 Как пользоваться

1. **База данных**: При первом добавлении ссылки из нового канала программа попросит указать название и количество баллов за одну активность.
2. **Наполнение**: Вставляйте ссылки во вкладке "Загрузка" по мере их появления.
3. **Генерация**:
   * Укажите цель (например, 300 баллов).
   * Нажмите "Сформировать".
   * В появившемся окне выберите новость (программа подскажет реальный профит с учетом лимита).
   * Получите готовый текст, разделенный на блоки 3.1 (Обычные) и 3.2 (Новости).

## 📊 Пример логики лимитов

| Цель отчета | Баллы новости | В отчет пойдет | Комментарий |
| :--- | :--- | :--- | :--- |
| **200** | 80 | **80** | Меньше лимита (100), идет в зачет полностью |
| **200** | 300 | **100** | Больше лимита, урезано до 50% от цели |
| **500** | 300 | **250** | Урезано до лимита 50% (250) |
